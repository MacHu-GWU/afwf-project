==============================================================================

Alfred Workflow Script Filter power tool.


.. _install:

Install
------------------------------------------------------------------------------

``afwf`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install afwf

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade afwf
